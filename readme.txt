== Description ==
Displays related blogs from our network at the end of each post and allows you to track click-through activity for each related blog. In turn, your blog can appear in these same places on related blogs, which helps boost traffic and increase inbound relevant links for better search engine rankings.

== Installation ==
1. Upload/unzip the 'alpenglo' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
